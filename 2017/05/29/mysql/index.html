navicat代码块
增删改查 –空格 等于//注释
增:
insert 表名(列名,列名,列名,列名) values(“值”,”值”,”值”,”值”,) //列名和值要对应;
INSERT information(NAME,sex,address) VALUES(“shabi115”,”man”,”beijing”)
删:
delete from 表名 where 列名=”值’ //where 判断列名的值符合条件 才删除
DELETE from information WHERE address=”beijing”
改:
update 表名 set 列名=”值” where 列名=”值”;
update information set address=’傻逼’ where nuber=2;
查: 多条件用and or
select 列名 from 表名 where 条件 group by 列名 order by asc/desc //asc和desc为正序和倒序
SELECT from information where name like “%ong%”; // %百分号为通配符 匹配用的
% 为通配 _表示只有一位 select后两个列名的话用,隔开
select 列名 as ‘小名’,列名 as “小名” from 表名 as ‘小名’
可以用as 直接查询后增加
order by 默认升序(主键) desc 降序 多个排序 用,隔开
分组查询:
流水帐;
SELECT id,gname,SUM(num) as num,price,SUM(num)price as sum from breakfast GROUP BY gname having 列名=”值(要用英文,中文不行)”;
//group by指的是分组 ;查询不能用通配符 要把每个列表都列出来,要插入一列的话,直接逗号后面+列名; 分组之后的筛选 用HAVING
count查询
SELECT count() from breakfast where gname=”baozi”;
返回当前条件的返回数量
多表连接
1.内连接:指取符合条件的数据
select from 表名,表名 where 列名=列名 and(条件) // 可以使用多个and
eg. SELECT FROM student as s,class as c where s.cid=c.id and c.cname LIKE ‘%前端%’;

子查询:
A.通过运算符查询
1.先查询出要的条件
SELECT height FROM student where stuname=”longwen”;
2.再用得到的数据和整个表格比较
SELECT FROM student where height>(SELECT height FROM student where stuname=”longwen”); //括号内的列名不能用;要用指定的列名
B.通过IN 实现子查询
SELECT from score WHERE stuid in (SELECT stuid FROM student where height>160);
//括号内的列名不能用;要用指定的列名;
in 可以直接用来查询
SELECT from score where stuid in(1,2) //stuid 为 1 和 2 的值,
SELECT from score where stuid not in(1,2) //not in 取反 stuid 为3,4;
limit 分页用的
SELECT FROM student LIMIT 3; //一个参数
SELECT FROM student LIMIT 0,2; //从零开始数 数两条 ,后面接着从2开始数也就是3,2;
公式: page=0,pagesize=2 上述limit的参数
page = ? pagesiez=2 ; limit (page-1)*pagesize, pagesize

代码实现约束:
alter table 表名
add constraint 约束名称 约束表达式
约束名称：
主键 PK列名
检查 CK列名
唯一 UK列名
默认 DF列名
外键 FK_列名

alter table course
   add constraint PK_cid primary key(cid)  --添加主键

alter table score
       add constraint CK_score check(score>0)  --添加检查约束，成绩必须大于0

   alter table course
    add constraint UK_cname unique(cname)      --添加唯一约束 【值不重复，但可以为NULL，一个表可以有多个此类型约束】
